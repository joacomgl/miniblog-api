# miniblog-api

Proyecto inicial `Node.js` + `Express` para la API del mini blog.

Estructura creada:

miniblog-api/
├── src/
│   ├── routes/
│   ├── controllers/
│   ├── services/
│   └── db.js
├── tests/
├── sql/
├── docs/
├── .env.example
├── .gitignore
├── package.json
└── README.md

Para inicio rápido:

1. Copiar `.env.example` a `.env` y ajustar valores.
2. `npm install` (si aún no se instalaron dependencias).
3. `npm start` para levantar el servidor.

El servidor escucha en el puerto definido en `PORT` del `.env`.
